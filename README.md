## English readme:

Please download python.exe and copy the following code into new_afk.py.

Please enter it in cmd: 
``` 
pip install numpy 
pip install opencv-python 
pip install keyboard 
pip install pillow 
pip install pyautogui 
pip install selenium 
pip install pytweening
```

If you get this error:

``` 
pip is not recognized as an internal or external command 
```
Then you need to uninstall python and reinstall it, then upgrade pip.

In general, downloading python from the official site (above) will not give you an error.

Download Ready.png, create a folder Images, put Ready.png under Images, and put new_afk.py side by side with the Images folder.

Ready.png: https://cdn.luogu.com.cn/upload/image_hosting/lnkukw72.png

After downloading, it will be webp by default, please go to CloudConvert to convert it to png.

File location: 
``` 
D:\ (or any other location)
->new_afk.py
->Images (this is a folder, please create it yourself)
->->Ready.png 
``` 
Please make sure the file extension of new_afk is .py.

Make sure your computer resolution is 1920 x 1080.

To run the program: open cmd in the directory where new_afk.py is located, type ```python.exe new_afk.py``` and press q to start the program.
